
# Project overview
A practice design implemented using TailwindCSS

## Technologies
1. HTML-5
2. Tailwind CSS
3. JS

## Live Demo
[Demo ](https://khaled6120.github.io/loopstudios-Tailwind-project/)

```
npm Install
```

Run Tailwind CLI

```
npm run watch
```
